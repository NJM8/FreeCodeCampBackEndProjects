Image Search

Add your search query to the end of the url above to search for images.

Example request: https://natethedevimagesearch.herokuapp.com/subaru

This will respond with a JSON object of images about Subaru's

You can get the next page of results like this: https://natethedevimagesearch.herokuapp.com/subaru?offset=2

You can see a history of searches by visiting: https://natethedevimagesearch.herokuapp.com/history
