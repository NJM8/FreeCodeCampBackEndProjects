URL Shortener

Visit the URL above with your destination appended to the end to receive a shortened URL you can use anywhere.

Example request: https://natethedevurlshortener.herokuapp.com/new/https://www.google.com

Example response: https://natethedevurlshortener.herokuapp.com/tinyLamb

Now visiting https://natethedevurlshortener.herokuapp.com/tinyLamb will send you to https://www.google.com

Please be sure to include the complete address as shown above.
